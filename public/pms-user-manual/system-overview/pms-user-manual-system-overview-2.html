产品介绍
产品定义
预案管理系统依托公共安全事件分级标准、风险隐患情况、处置阶段划分、应急处置知识等行业经验，借助预案大脑、预案编制、预案推演和预案统计软件功能，帮助用户建立健全应急预案体系，为灾害发生后的高效协同处置提供辅助决策支撑。
产品主要服务对象
政府相关应急管理部门、企事业单位应急管理部门。
产品业务范围
预案大脑、预案管理、预案推演、预案统计。
