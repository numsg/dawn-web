关键词定义
为了便于您更好的理解系统，将对一些关键字进行定义解释，具体请参见下表。
关键词 定义
预案
指在灾害发生之前对灾害应对过程中综合协调和应急处置所依赖的关键任务和所需要的应急保障资源进行建设，为灾害快速高效协同响应提供权威支撑。
指导预案
针对一类或多类灾害事件制定，用来指导灾害分级和响应分级、组织机构责任划分、灾情预警预报与信息共享、应急响应、灾后救助与重建、应急保障资源准备等工作的纲领性文件。
执行预案
针对特定类型、规模的事件制定，详细分解关键任务和应急保障资源需求的可执行性的任务计划。
预案推演
为了验证预案的完备性，在过往案例事件情景中或假定事件情景中依据事件的基本信息（如伤亡人口、受影响人口、经济损失、影响面积以及现场实时监测信息等），按照处置阶段对事件应参与的人员、执行的任务和依赖的应急资源进行详细的定义，并在模拟事件情景中预演响应过程、判断预案是否完备。
预案大脑
预案大脑存储预案知识，包括灾害损失、灾害级别、响应级别、处置阶段、组织机构、应急资源、处置任务、规则库、元件库、预案模型等。
