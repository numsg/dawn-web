预案模型导出
系统支持将每个预案附属的预案模型导出，即包括预案文件、此预案使用的规则库、此预案使用的数据源、此预案使用的元件库进行打包导出。
具体操作如下：
1. 在主菜单中选择“预案管理→预案模型”菜单，进入“预案模型”页面。
2. 在“导出知识模型”页签下，勾选需要导出的知识内容。
可以沟通具体的方框选择某个数据，可以通过勾选分类前的圆圈选择某类或全部数据
3. 单击〖导出知识模型〗按钮，弹出“导出知识模型”窗口。
4. 输入模型名称以及事件类型。
5. 单击〖生成〗按钮，自动生成密钥。
单击〖复制〗按钮，将该密钥复制并保存。
6. 单击〖保存〗按钮，将该模型保存至本地。
说明:
导出后的知识模型是经过加密的，需要将该密钥保存，若要再次导入系统需要输入该密钥。
