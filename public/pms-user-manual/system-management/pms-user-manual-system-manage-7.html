用户管理
用户管理用于系统用户的日常维护管理。系统支持对系统用户进行查询、添加、修改、为用户添加角色、重置密码、删除操作。
在主菜单中选择“系统管理→用户管理”菜单，内容显示区显示系统用户。
具体操作如下表所示：
序号 操作项 操作步骤
1 查询用户
在搜索输入框输入用户名称关键字。
显示符合查询条件的记录。
2 添加用户
1. 在“用户管理”页面，单击左上角〖添加用户〗按钮，界面右侧显示“添加用户”区域。
2. 输入用户信息。
3. 单击右上角〖保存〗按钮，保存该用户。
说明:
用户信息添加成功后，需要为该用户添加角色，赋予该用户某一角色的权限。
3 为用户添加角色
用户关联角色才能登录系统进行相关业务操作，且可关联一个或多个角色。
1. 在“用户管理”页面选择待添加角色的用户。
2. 在“用户角色”区域单击〖添加〗按钮，界面右侧显示“添加角色”区域。
3. 在“添加角色”区域勾选一个或多个待添加的角色复选框。
4. 单击〖保存〗按钮，系统提示角色添加成功。
2 修改用户
1. 在“用户管理”页面选择待修改的用户。
2. 修改该用户信息。其中，“用户名”不可修改。
3. 修改用户角色信息。
•  单击〖添加角色〗按钮，可打开“添加角色”区域，继续修改该用户已添加的角色信息。
•  单击各个角色右侧按钮，可为该用户删除已添加的角色。
4. 单击右上角〖保存〗按钮，保存修改信息。
4 重置用户密码
1. 在“用户管理”页面，选择待重置密码的用户。
2. 单击〖重置密码〗按钮。
界面提示密码重置成功。重置后的密码为后台配置的“123456”。
5 删除用户
1. 选择待删除的用户，单击用户右侧按钮，弹出删除确认对话框
2. 单击〖确认〗按钮，删除该用户。
