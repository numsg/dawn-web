角色管理
用户结合使用需要进行角色添加配置操作，并为不同的角色关联所需的资源，关联成功后，该角色用户在进行系统登录时，拥有对配置资源的操作权限。
在主菜单中选择“系统管理→角色管理”菜单，内容显示区角色管理界面。
具体操作如下表所示。
序号 操作项 操作步骤
1 查询角色信息
在搜索输入框输入角色名称关键字。
显示符合查询条件的记录。
2 添加角色
1. 在“角色管理”页面，单击〖添加角色〗按钮，界面右侧显示“添加角色”区域。
2. 输入待添加的角色信息。
3. 单击右上角〖保存〗按钮，保存该角色。
说明:
新增角色成功后，需要为该角色关联资源。新增角色保存后才可为其关联资源权限。
标记“*”项为必填项。
3 为角色关联资源
1. 在“角色管理”页面，选择待添加资源的角色。
2. 在下方“已关联权限资源”区域单击〖关联〗按钮。
3. 勾选需要添加的权限资源复选框。
4. 单击〖保存〗按钮，系统提示角色关联资源成功
说明:
（1）资源管理菜单下的“增加资源”、“删除资源”、“修改资源”等按钮资源无法被角色关联，仅内部超级管理员拥有这些按钮权限
（2）权限资源包含“功能资源”和“API资源”2个页签。
3 修改角色
1. 在“角色管理”页面，单击选择待修改的角色。
界面右侧显示“角色详情”区域。
2. 修改该角色信息。
3. 修改角色关联资源权限。
•  单击〖关联权限资源〗按钮，
•  单击各个主菜单模块右侧的按钮，可修改该角色关联的当前主菜单所包含的权限资源。
•  单击各个主菜单模块右侧按钮，可删除该角色关联的当前主菜单所包含的权限资源
4. 单击右上角〖保存〗按钮，保存修改信息。
4 删除角色
1. 选择待删除的角色，单击〖 〗按钮，弹出删除确认对话框
2. 单击〖确认〗按钮，删除该角色。
说明:
不能删除被用户关联的角色信息。
