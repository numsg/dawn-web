系统配置
在主菜单中选择“系统管理→系统配置”菜单，进入“系统配置”页面。系统管理员通过系统配置功能实现对系统的一些自定义设置。
全局配置
以“全局配置”为例重点介绍，其余配置可参考全局配置。
序号 操作项 操作步骤
1
新增配置
1. 在配置类型列表单击〖全局配置〗按钮，右侧对应展示配置页面。
2. 单击按钮，展示新增窗口。
3. 输入名称、key、值以及描述。
4. 单击〖保存〗按钮。
2
修改配置
以“退出登录时间配置”为例介绍。
1. 在配置类型列表单击“全局配置→退出登录时间配置”，右侧对应展示配置页面。
2. 单击按钮，进入编辑状态。
3. 设置时间参数
4. 单击，保存修改。
3
删除配置
以“退出登录时间配置”为例介绍。
1. 在配置类型列表单击“全局配置→退出登录时间配置”，右侧对应展示配置页面。
2. 单击按钮，系统弹出删除确认提示。
3. 单击〖确认〗按钮。
业务数据源配置
对业务数据源的标识ID进行配置操作。包括：新增业务数据源类型、编辑以及删除已有数据源等。
元件类型配置
对元件类型进行配置。
规则类型配置
对规则类型进行配置。
元件工具配置
进行元件工具相关配置。
资源管理系统配置
进行是否启动资源管理系统设置。
资源系统数据源配置
对资源系统数据源中的标识ID进行配置。
批注功能配置
进行批注相关功能配置，包括：保存批注URL，查询远端元件URL、是否查询远端元件设置。
GIS配置
进行是否启动arcGIS服务设置。
